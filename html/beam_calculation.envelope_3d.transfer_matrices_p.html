Parameters:

  • gamma_in (float) ‚Äì Lorentz gamma at entry of drift.

  • n_steps (int, optional) ‚Äì Number of integration steps. The number of integration steps has no influence on the results. The default is one. rk4(u, du, x, dx)ÔÉÅ

4-th order Runge-Kutta integration.

This function calculates the variation of u between x and x+dx. Warning: this is a slightly modified version of the RK. The k_i are proportional to delta_u instead of du_dz.

Parameters:

  • u (np.array) ‚Äì Holds the value of the function to integrate in x.

  • du_dx (function) ‚Äì Gives the variation of u components with x.

  • x (real) ‚Äì Where u is known.

  • dx (real) ‚Äì Integration step.

Returns:

real
